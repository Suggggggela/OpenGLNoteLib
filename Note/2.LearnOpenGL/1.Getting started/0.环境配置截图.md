依赖文件夹中分为两个文件夹，分别存放.h头文件(Include)和.lib库文件(Lib)
其中Cherno教程中的命名为Dependencies

## 路径配置Note：
  在配置include和lib文件夹时，可以使用相对路径的方法进行配置：
    首先，`$(SolutionDir)`这个宏代表当前solution的位置

`../`可以指定到当前目录的上级目录



这里项目中的依赖文件夹在项目目录的外三层的OpenGLReference文件夹中，那么对应的路径应该是这样的

`$(SolutionDir../../../OpenGLReference/xxxxx)`

## 项目全局配置

### 创建PropertySheet

在ViewPort-Other Windows - Property Manager中可以打开属性管理器



在任意目录处，右键Add New Project Property Sheet

打开后设置相关属性 



记得勾选Edito - Inherit from xxxxx

```
$(SolutionDir)Dependencies/includes;

$(SolutionDir)Dependencies/lib;

glfw3.lib;opengl32.lib;kernel32.lib;user32.lib;gdi32.lib;winspool.lib;comdlg32.lib;advapi32.lib;shell32.lib;ole32.lib;oleaut32.lib;uuid.lib;odbc32.lib;odbccp32.lib;%(AdditionalDependencies)
```

设置之后保存

### 应用到新Project

在新创建的Project上，右键，Add Extensting Property Sheet即可
