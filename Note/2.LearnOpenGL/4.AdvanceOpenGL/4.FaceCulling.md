面剔除是经典的优化手段，比如Unity常写的cullback，cullfront就是今天这节的内容
我们今天讨论一下OpenGL中的面剔除

## 1.环绕顺序
环绕顺序对于面剔除是很好的前提理解，我们可以看到逆时针和顺时针的环绕顺序表示如下：
![459x267](assets/4.FaceCulling_image_1.png)

```cpp
float vertices[] = 
{ 
// 顺时针 
vertices[0], // 顶点1 
vertices[1], // 顶点2 
vertices[2], // 顶点3 

// 逆时针 
vertices[0], // 顶点1 
vertices[2], // 顶点3 
vertices[1] // 顶点2 
};

```

在定义完面的环绕顺序后，OpenGL就可以通过这个确定一个面试正向还是背向三角形了（默认情况下，逆时针顺序为正向三角形，接下来都以这个为前提）
当你从观察者视角看一个面的时候，如果这个面是正向的，那么它的环绕顺序应该是逆时针的：
![466x251](assets/4.FaceCulling_image_2.png)

如果这时有一个面是顺时针的，那么就代表它是背向的

## 2.面剔除
在知道如何获取一个面是正向还是背向之后，我们看看OpenGL是如何进行面剔除的

```cpp
glEnable(GL_CULL_FACE);
glCullFace(GL_FRONT);//GL_BACK为默认，GL_FRONT,GL_FRONT_AND_BACK为可选

//如果你要自定义那个顺序是正向面，可以这样：
glFrontFace(GL_CCW);//默认=CCW为逆时针，CW=顺时针
```
