物体表面呈现的颜色是经过光的照射后，反射出未吸收的能量的结果

以简单的伪代码为例，我们可以这么看

```cpp
glm::vec3 lightColor(0.0f, 1.0f, 0.0f); 
glm::vec3 toyColor(1.0f, 0.5f, 0.31f); 
glm::vec3 result = lightColor * toyColor; // = (0.0f, 0.5f, 0.0f);

//toyColor = objectColor
//当灯光颜色为绿色时，它照射到物体表面，红色和蓝色都被物体吸收了，呈现的就是物体绿通道的内容

```


如果要创造一个有光照的场景，我们可以有以下思路：
- 光源
    - 以一个cube作为光源，shader中输出颜色作为LightColor
    - cube的位置作为LightPosition
- 物体
    - 物体shader接受光源的参数作为输入，进行着色



当然这节我们只做一个简单的描述，后续我们会进行光照的计算
