
## 一、什么是混合
渲染时不是简单的一层图层，我们可以用PS的思路来理解
当涉及到多层的时候，加上透明度，就是我们常说的Blending

## 二、OpenGL中的Blending

### 1.开启混合
OpenGL默认是关闭混合模式的，我们需要通过下边的指令打开混合模式

```cpp
glEnable(GL_BLEND);
glDisable(GL_BLEND);
```

### 2.混合方程
混合方程是指告诉OpenGL如何对源颜色和目标颜色进行混合
- src ，默认为1
- dest，默认为0

```cpp
glBlendFunc(src,dest);
```

### 3.混合模式
混合指定src和dest的混合方式，默认是Add
```cpp
glBlendEquation(mode);
```

### 4.在OpenGL中使用Blending

```cpp
//BlendMode -> AlphaBlend
	GLCall(glEnable(GL_BLEND));
	GLCall(glBlendFunc(GL_SRC_ALPHA, GL_ONE_MINUS_SRC_ALPHA));

```
