
## 一、什么是投影
投影是3D世界投影到2D屏幕的方法

## 二、投影矩阵
投影矩阵可以将模型的顶点转换投影到NDC空间（归一化设备空间：Normalized Device Coordinates）
归一化意味着每个轴的值都映射到了（-1,1），所以它和窗口的分辨率并不相关

### 2.例子中的正交投影
我们先在shader中将fragment输出改为白色
```cpp
//例子中的屏幕分辨率
window = glfwCreateWindow(640, 480, "LearnOpenGL", NULL, NULL);
//正价投影矩阵
//这个矩阵将坐标系的范围设置为[-2,2]和[-1.5,1.5]，z轴的范围是[-1,1]，这个矩阵是一个正交投影矩阵
glm::mat4 projection = glm::ortho(-2.0f, 2.0f, -1.5f, 1.5f, -1.0f, 1.0f);
```
![301x246](assets/19.ProjectionMartix_image_2.png)

当我们把窗口分辨率改为960x540，结果如下：
![394x235](assets/19.ProjectionMartix_image_3.png)
这样被拉伸的原因是：投影矩阵没变，但我们使用了不同的窗口分辨率

为了理解这到底发生了什么，我们可以这样做：
```cpp
//将顶点的位置拉宽
float positions[] =
{
	100.0f,100.0f,0.0f,0.0f,
	200.0f,100.0f,1.0f,0.0f,
	200.0f, 200.0f,1.0f,1.0f,
	100.0f, 200.0f,0.0f,1.0f
};

//修改正交投影矩阵为屏幕的宽高
glm::mat4 projection = glm::ortho(0.0f, 960.0f, 0.0f, 540.0f, -1.0f, 1.0f);
```
