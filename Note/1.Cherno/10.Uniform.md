## 一、什么是Uniform
Uniform是GPU从CPU端获取数据的一种方式
## 二、uniform的简单使用
下边的例子中，我们使用uniform类型变量替换fragment输出的颜色，并通过CPU端设置颜色

shader中添加uniform变量
```cpp
//ShaderSourceCode
#shader fragment
#version 330 core
out vec4 color;
uniform vec4 u_Color;
void main()
{
	color = u_Color;
};
```

cpp中使用uniform变量
```cpp
int location = glGetUniformLocation(shader, "u_Color");
glUniform4f(location, r, 0.3f, 0.8f, 1.0f);//根据类型设置uniform变量的值
```

如果想做点好玩的，比如让画出的颜色改变，可以这样
```cpp
//在glfw创建window之后，设置刷新频率和屏幕一直
glfwSwapInterval(1);//设置交换间隔为1，表示每帧交换一次缓冲区，和显示器的刷新率一致


//RenderLoop外设置初始R通道值，并声明一个增量
float r = 0.0f;
float increament = 0.05f;
while(!glfwWindowShouldClose(window))
{
    glUniform4f(location, r, 0.3f, 0.8f, 1.0f);
    GLCall(glDrawElements(GL_TRIANGLES, 6, GL_UNSIGNED_INT, nullptr));
    
    if(r>1.0f)
    {
    	increament = -0.05f;
    }
    else if (r < 0.0f)
    {
    	increament = 0.05f;
    }
    r += increament;
}
```
