
## 一、什么是MVP矩阵
MVP三个字母分别代表Model，View，Projection三个矩阵
他们用来对顶点进行各种空间的变化

对于应用矩阵，我们需要注意：
- MVP矩阵是: M * V * P
- OpenGL是列矩阵排序
- DX是行矩阵排序

## 二、使用MVP矩阵
### 1.MVP矩阵代表着什么
- M矩阵代表模型的位置，旋转，变换
- V矩阵代表摄像机的位置、旋转、缩放
- P矩阵负责将其他空间的坐标转换为NDC空间

### 2.使用MVP
#### ViewMatrix
OpenGL中实际上没有Camera，那么是怎么做到看起来有的呢？
答案是：对被Camera看到的物体做反向操作
我们来看一个例子：
```cpp
//我们想模拟相机向右移动100个float的情况，所以V矩阵向左移动100个float
glm::mat4 view  = glm::translate(glm::mat4(1.0), glm::vec3(-100, 0, 0));
glm::mat4 mvp = projection * view;

shader.SetUniformMat4f("u_MVP", mvp);
```

#### ModelMatrix
这里我们先不做太多关于3D的讨论，说简单点
比如我现在想通过模型矩阵修改渲染出正方形的位置，可以吗？
当然可以：
```cpp
glm::mat4 model = glm::translate(glm::mat4(1.0), glm::vec3(200, 200, 0));//Model空间向上移动200，向右移动200
glm::mat4 mvp = projection * view * model;
```
